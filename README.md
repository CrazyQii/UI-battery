# UI-battery

该项目涉及技术：
1. 数据库：MongoDB
2. 前端：Jquery， Bootstrap
3. 后端：Nodejs， nunjucks， express

实现的功能：
1. 系统实时显示已记录的公交车公司所拥有的未监控和已监控的公交车数量；
2. 添加、删除、查询公交车的详细信息；
3. 查询公交车信息并在地图中显示；

未解决功能：
1. 用户注册登录功能未能实现
2. 公交车电量查询详细信息功能未能实现

关键技术：
o	Nodejs 和 express 的搭建；
o	Ajax 请求；
o	MongoDB 数据库的 CURD；
o	对数据的分页查询；
o	Mockjs 数据的模拟；
o	百度地图 API 的引用；
